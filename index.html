1953.live

Share your stories.

1953 is a web-first marketplace app designed to help book lovers exhange books, make friends, and share stories.

The app was developed in-house from conceptualisation, through to development and production.

1953 is built on an MVC framework with a stack including Ruby on Rails, HTML, CSS, JS, Bootstrap, Figma, Postgre SQL and more.

1953's features include a live chat to arrange swaps, a search utilising Google Books API, and geolocation using the Mapbox API, allowing users to see the closest books to them.

Island Hopper

The island experience made easy.

Island Hopper is a MVP/proof-of-concept app developed to allow users to create profiles, rent their own islands, and request stays at other islands.

Island Hopper features a location search built using the Algolia Places API along with an interactive map using Mapbox API.

The app is built on an MVC framework with a stack including Ruby on Rails, HTML, CSS, JS, Bootstrap, Postgre SQL and more.

Fresh Picks

Organic vegboxes delivered to your door.

Fresh-Picks is the e-commerce site for a subscription-based organic farm/ vegbox service based in the UK.

Fresh Picks features Stripe API implementation for a seamless payment experience, along with real-time updates from the company Instagram page.

The app is built on an MVC framework with a stack including Ruby on Rails, HTML, CSS, JS, Bootstrap, Postgre SQL and more.
