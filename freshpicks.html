Fresh Picks

Organic vegboxes delivered to your door.

Fresh-Picks is the e-commerce site for a subscription-based organic farm/ vegbox service based in the UK.

Fresh Picks features Stripe API implementation for a seamless payment experience, along with real-time updates from the company Instagram page.

The app is built on an MVC framework with a stack including Ruby on Rails, HTML, CSS, JS, Bootstrap, Postgre SQL and more.
